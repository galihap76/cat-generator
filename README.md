# 🐈 Cat Generator

Cat generator, buat generate gambar kucing secara acak menggunakan React.js. Hanya iseng membangun web ini.

# Install

- ```
  git clone https://github.com/galihap76/cat-generator.git
  ```
- ```
  cd cat-generator
  ```
- ```
  npm install
  ```
- ```
  npm run dev
  ```
